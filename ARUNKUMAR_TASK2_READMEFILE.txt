# Web Technologies Detection Script

This Python script is designed to detect the presence of various web technologies on a list of websites provided in a CSV file. It can identify technologies such as jQuery, WordPress, Joomla, and more by analyzing the source code of the websites.

## Prerequisites

- Python 3.x
- Required Python packages (can be installed via pip):
  - requests
  - beautifulsoup4
  - pandas

## Usage

1. Clone this repository or download the script (ARUNKUMAR_TASK2_PROGRAM.py) to your local machine.

2. Prepare your input CSV file, order.csv, which should contain a list of website URLs in a single column. You can use the provided order.csv as a sample.

3. Run the script using the following command:ARUNKUMAR_TASK2_PROGRAM.py
4. The script will start processing the URLs and detecting web technologies. The results will be saved to an output CSV file named arun_Task2_Output.csv.

## Error Handling

- If an error occurs while processing a website, the error message will be logged in the scraping.log file.
