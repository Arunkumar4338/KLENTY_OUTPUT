# Microsoft Partner Directory Scraper

This Python script is designed to scrape data from the Microsoft Partner Directory on the AppSource platform. It extracts information about Microsoft partners, including partner names, regions, short descriptions, and the href of the "Contact Me" button link. The scraped data is then saved to a CSV file for further analysis or use.

## Prerequisites

- Python 3.x
- Required Python packages (can be installed via pip):
  - requests
  - pandas

## Usage

1. Clone this repository or download the script (microsoft_partner_scraper.py) to your local machine.

2. Run the script using the following command:

   ```bash
   python microsoft_partner_scraper.py
