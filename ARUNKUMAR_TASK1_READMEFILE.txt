# Contact Information Scraper

This Python script is designed to scrape contact information, including emails and phone numbers, from a list of company websites. The information is then stored in a CSV file for further analysis or use.

## Prerequisites

- Python 3.x
- Required Python packages (can be installed via pip):
  - requests
  - beautifulsoup4

## Usage

1. Clone this repository or download the script (ARUNKUMAR_TASK1_PROGRAM.py) to your local machine.

2. Prepare the list of company details:
   - You can modify the companies list in the script with your own list of company names and their respective website URLs.
   - Each company should be represented as a tuple with the company name and the website URL.

3. Run the script using the following command:
   
   ```bash
   python contact_info_scraper.py
